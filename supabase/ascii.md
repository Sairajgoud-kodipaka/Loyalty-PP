## DESIGN PRINCIPLES USED

### 1. Visual Hierarchy
- Large numbers for important data (points, amounts)
- Clear section headers with icons
- Color coding: Green (available), Yellow (pending), Red (expiring)

### 2. User Flow
- Maximum 3 clicks to complete any task
- Clear "Back" navigation on every page
- Confirmation modals for critical actions

### 3. Responsive Design
- Desktop: Multi-column layout with side-by-side cards
- Mobile: Single column, stacked elements
- Touch targets: Minimum 44x44px on mobile

### 4. Accessibility
- High contrast text
- Icon + text labels
- Clear error messages
- Keyboard navigation support

### 5. Performance
- Real-time calculations displayed
- Minimal page reloads
- Optimistic UI updates
- Loading states for async operations
